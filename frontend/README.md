# Psychological Mirror

Psychological Mirror is an AI-assisted journaling system that extracts emotional signals and structured reflections from free-form entries using LLM pipelines.

## Stack

Frontend: React
Backend: Express
Database: SQLite
AI: OpenAI API

v0.1 — Extractor engine + journaling pipeline
